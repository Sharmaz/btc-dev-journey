# Communities

Bitcoin development is collaborative. These are the places where protocol discussions happen, code gets reviewed, and developers connect.

## Discussion Forums

### Delving Bitcoin
**Link:** [delvingbitcoin.org](https://delvingbitcoin.org/)

The primary forum for technical Bitcoin discussions. Topics include protocol proposals, implementation details, research, and development. Created to replace the bitcoin-dev mailing list for long-form discussions.

### Bitcoin StackExchange
**Link:** [bitcoin.stackexchange.com](https://bitcoin.stackexchange.com/)

Q&A site for Bitcoin-specific technical questions. High-quality answers from experienced developers. Great for finding answers to specific implementation questions.

### BitcoinTalk Development Forum
**Link:** [bitcointalk.org/index.php?board=6.0](https://bitcointalk.org/index.php?board=6.0)

The original Bitcoin forum. The Development & Technical Discussion board still has activity, though most protocol discussion has moved to Delving Bitcoin.

## Mailing Lists

### bitcoin-dev
**Link:** [groups.google.com/g/bitcoindev](https://groups.google.com/g/bitcoindev)

The official mailing list for Bitcoin protocol development. BIP proposals, soft fork discussions, and protocol changes are announced here. Essential reading for protocol developers.

### lightning-dev
**Link:** [lists.linuxfoundation.org/pipermail/lightning-dev](https://lists.linuxfoundation.org/pipermail/lightning-dev/)

Technical discussion about Lightning Network protocol development, BOLT specifications, and implementation details.

## IRC / Chat

### Libera Chat IRC
**Link:** [libera.chat](https://libera.chat/) (use any IRC client)

| Channel | Topic |
|---------|-------|
| `#bitcoin-core-dev` | Bitcoin Core development |
| `#bitcoin-core-pr-reviews` | PR review discussions |
| `#bitcoin-wizards` | Advanced protocol research |
| `#lightning-dev` | Lightning protocol development |
| `#bitcoin-builds` | Build system and reproducible builds |

IRC is where real-time Bitcoin Core development happens. Many core contributors are active daily.

### Nostr
**Link:** Various clients ([nostr.com](https://nostr.com/))

Growing Bitcoin developer community on Nostr. Follow Bitcoin developers, join topic-based groups, and discuss development. Many Bitcoiners are active on Nostr as it aligns with Bitcoin's decentralization values.

## GitHub

### Bitcoin Core
**Link:** [github.com/bitcoin/bitcoin](https://github.com/bitcoin/bitcoin)

Issues and pull requests are the primary venue for Bitcoin Core development. Watch the repository to follow development.

### Bitcoin Improvement Proposals
**Link:** [github.com/bitcoin/bips](https://github.com/bitcoin/bips)

All BIPs are tracked here. Read open PRs to follow proposed changes to the protocol.

### Lightning Specifications
**Link:** [github.com/lightning/bolts](https://github.com/lightning/bolts)

BOLT specifications and discussions about Lightning Network protocol changes.

## Getting Involved

1. **Start by lurking** — Read discussions, understand the culture
2. **Ask questions** — StackExchange is the best place for specific questions
3. **Join PR Review Club** — [bitcoincore.reviews](https://bitcoincore.reviews/) every Wednesday
4. **Attend a meetup** — In-person connections are invaluable
5. **Contribute** — Review code, write documentation, report bugs
