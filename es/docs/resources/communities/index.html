Comunidades

El desarrollo de Bitcoin es colaborativo. Estos son los lugares donde ocurren las discusiones de protocolo, se revisa el código y los desarrolladores se conectan.

Foros de Discusión​

Delving Bitcoin​

Link: delvingbitcoin.org

El foro principal para discusiones técnicas de Bitcoin. Los temas incluyen propuestas de protocolo, detalles de implementación, investigación y desarrollo. Creado para reemplazar la lista de correo bitcoin-dev para discusiones de formato largo.

Bitcoin StackExchange​

Link: bitcoin.stackexchange.com

Sitio de preguntas y respuestas para cuestiones técnicas específicas de Bitcoin. Respuestas de alta calidad de desarrolladores experimentados. Excelente para encontrar respuestas a preguntas específicas de implementación.

BitcoinTalk Development Forum​

Link: bitcointalk.org/index.php?board=6.0

El foro original de Bitcoin. El tablero de Development & Technical Discussion aún tiene actividad, aunque la mayoría de las discusiones de protocolo se han movido a Delving Bitcoin.

Listas de Correo​

bitcoin-dev​

Link: groups.google.com/g/bitcoindev

La lista de correo oficial para desarrollo del protocolo Bitcoin. Propuestas de BIP, discusiones de soft fork y cambios al protocolo se anuncian aquí. Lectura esencial para desarrolladores de protocolo.

lightning-dev​

Link: lists.linuxfoundation.org/pipermail/lightning-dev

Discusión técnica sobre desarrollo del protocolo Lightning Network, especificaciones BOLT y detalles de implementación.

IRC / Chat​

Libera Chat IRC​

Link: libera.chat (usa cualquier cliente IRC)

CanalTema
#bitcoin-core-devDesarrollo de Bitcoin Core
#bitcoin-core-pr-reviewsDiscusiones de revisión de PRs
#bitcoin-wizardsInvestigación avanzada de protocolo
#lightning-devDesarrollo del protocolo Lightning
#bitcoin-buildsSistema de build y builds reproducibles

IRC es donde ocurre el desarrollo de Bitcoin Core en tiempo real. Muchos contribuidores core están activos diariamente.

Nostr​

Link: Varios clientes (nostr.com)

Comunidad creciente de desarrolladores Bitcoin en Nostr. Sigue a desarrolladores Bitcoin, únete a grupos temáticos y discute desarrollo. Muchos Bitcoiners están activos en Nostr ya que se alinea con los valores de descentralización de Bitcoin.

GitHub​

Bitcoin Core​

Link: github.com/bitcoin/bitcoin

Issues y pull requests son el lugar principal para el desarrollo de Bitcoin Core. Observa el repositorio para seguir el desarrollo.

Bitcoin Improvement Proposals​

Link: github.com/bitcoin/bips

Todos los BIPs se rastrean aquí. Lee los PRs abiertos para seguir los cambios propuestos al protocolo.

Lightning Specifications​

Link: github.com/lightning/bolts

Especificaciones BOLT y discusiones sobre cambios al protocolo Lightning Network.

Comunidades Regionales​

Bitcoin Guadalajara​

Link: btcgdl.com

Comunidad local de Bitcoin en Guadalajara, México.

Encontrando Comunidades Locales​

  • Grupos de meetup de Bitcoin — Busca en meetup.com para tu ciudad
  • Conferencias Bitcoin — Bitcoin++, Adopting Bitcoin, Baltic Honeybadger, TABConf
  • Hackathons — Muchas conferencias incluyen hackathons para desarrolladores
  • PlanB Network — planb.network lista comunidades en todo el mundo

Cómo Involucrarte​

  1. Empieza observando — Lee discusiones, entiende la cultura
  2. Haz preguntas — StackExchange es el mejor lugar para preguntas específicas
  3. Únete al PR Review Club — bitcoincore.reviews cada miércoles
  4. Asiste a un meetup — Las conexiones en persona son invaluables
  5. Contribuye — Revisa código, escribe documentación, reporta bugs
