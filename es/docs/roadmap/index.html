Roadmap del Desarrollador Bitcoin

Haz clic en cualquier nodo para aprender más sobre el tema y encontrar enlaces a la documentación correspondiente.

Cómo Usar Este Roadmap​

  1. Comienza con los Fundamentos — Todos inician con las mismas bases universales: prerequisitos, conceptos básicos de Bitcoin, arquitectura, red P2P y desarrollo básico
  2. Elige Tu Track — Una vez que tengas una base sólida, elige la especialización que más te entusiasme
  3. Profundiza — Cada track tiene su propia progresión desde temas básicos hasta avanzados
  4. Explora — Muchos conceptos se comparten entre tracks, así que explora libremente

Tracks de Especialización​

TrackEnfoqueConstruirás
