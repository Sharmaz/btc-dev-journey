## Cómo Usar Este Roadmap

1. **Comienza con los Fundamentos** — Todos inician con las mismas bases universales: prerequisitos, conceptos básicos de Bitcoin, arquitectura, red P2P y desarrollo básico
2. **Elige Tu Track** — Una vez que tengas una base sólida, elige la especialización que más te entusiasme
3. **Profundiza** — Cada track tiene su propia progresión desde temas básicos hasta avanzados
4. **Explora** — Muchos conceptos se comparten entre tracks, así que explora libremente

## Tracks de Especialización

| Track | Enfoque | Construirás |
|-------|---------|------------|
| **Desarrollador de Protocolo** | Internos de Bitcoin Core, consenso, BIPs | Contribuciones al core, mejoras al protocolo |
| **Desarrollador de Aplicaciones** | Librerías, wallets, sistemas de pago | Wallets, procesadores de pago, exchanges |
| **Desarrollador de Infraestructura** | Exploradores de bloques, indexadores, APIs | Exploradores, plataformas de análisis, infraestructura de nodos |
| **Desarrollador de Minería** | Software de minería, pools, hardware | Pools de minería, firmware, herramientas de optimización |

¿Listo para comenzar? Dirígete a la sección de [Prerequisitos](/docs/fundamentals/prerequisites).
